OVERVIEW
--------
This folder contains 50 Moodle XML quiz files organised into two series:

  Part B  (B01–B25)  —  25 files  —  Single-correct MCQs
  Part C  (C01–C25)  —  25 files  —  Multiple-correct MCQs

Each file covers one topic and contains 10–20 CSIR NET style questions with
full worked solutions in generalfeedback (HTML + LaTeX MathJax).

MOODLE FORMAT NOTES
-------------------
  Question type   : multichoice
  Category path   : $course$/CSIR NET Mathematics/Similar Questions/Part B (or C)/<Topic>
  Answer numbering: abc
  Shuffle answers : true (Part B),  false (Part C)

  Part B  →  single="true",  penalty=0.25,  correct fraction=100,  wrong=0
  Part C  →  single="false", penalty=0,     fractions split equally among correct answers
             (e.g. 3 correct → 33.33333 each;  4 correct → 25 each;  2 correct → 50 each)

  IMPORT INSTRUCTIONS (Moodle)
================================================================================

  1. Log in to Moodle as admin / course teacher
  2. Go to: Course → Question Bank → Import
  3. Select format: "Moodle XML"
  4. Upload the desired XML file(s)
  5. Questions will appear under the category path embedded in the XML:
       $course$/CSIR NET Mathematics/Similar Questions/Part B/<Topic>
       $course$/CSIR NET Mathematics/Similar Questions/Part C/<Topic>
  6. Create a Quiz activity → Add questions from bank → filter by category
